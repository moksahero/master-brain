![Legends SEO Dungeon banner](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/9c4b31bd1c384d7a9791ecdeaf4a32fdcf83189689bf43d48a3248b682b5004a.jpg)

## Legends SEO Dungeon

> Private Codex-first SEO audits turned into visible quests, priorities, and fixes.

Legends SEO Dungeon is the private Legends build of the SEO Dungeon workflow. It runs on the v2.2 SEO engine, keeps the browser game bridge Codex-first, and now includes structured Codex remote control for the Guild Ledger loop.

The bundled skills, scripts, and portable agent prompts remain readable by compatible terminal-agent workflows, but the default packaged experience is Codex.

Use it when you want SEO issues to feel concrete instead of buried in a report: the audit becomes a game board, problems become quests, and the next action is easier to see.

## Best For

- SEO audits that need to become action, not just notes.
- Turning technical and content issues into visible priorities.
- Client-facing or team-facing SEO cleanup work.
- Codex-first private SEO workflows with portable skill files.
- Fast navigation from problem discovery to fix planning.

## Current Release

- Version: v2.2.16 Structured Remote Control
- Engine: Codex SEO v2.2
- Skill surface: 25 sub-skills, 18 portable Markdown agent prompts, and 23 Codex profiles.
- Default profiles: Warrior deep / xhigh, Samurai balanced / high, Knight fast / medium.

## What Changed In v2.2.16

- Codex can drive browser-owned setup, Gate resume, Hall selection, Battle attack, queued prompt steering, stop, clear, and vanquish through structured local events.
- The Guild Ledger mirrors command receipts, UI results, active operation state, and replayable session history between the browser and helper CLI.
- Browser and desktop proof recorders now capture video, screenshots, manifests, session logs, ledger transcripts, CLI receipts, and bridge output for recursive testing.
