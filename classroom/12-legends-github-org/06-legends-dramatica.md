## Legends Dramatica

> Longform expressive TTS production with DramaBox, now with an agent skill suite for planning, QA, rerenders, and production memory.

Legends Dramatica is a private production desk for longform expressive TTS, DramaBox voice cloning, and batch WAV rendering.

It turns DramaBox from a one-prompt demo surface into a production workflow: paste a longer script, split it into scene-sized modules, generate consistent expressive takes, review clips, reroll weak parts, and stitch approved WAVs into a final track.

The current v0.3.2 release ships the cross-agent skill suite for Codex, Claude, and Gemini, plus the Nicole starter voice reference and a cleaner refresh installer. Agents now have a shared workflow for intake, voice setup, production planning, render QA, targeted rerenders, transcript/alignment prep, and durable vault memory.

## Best For

- Dramatic narration and scene-based audio work.
- DramaBox voice-reference and voice-cloning workflows.
- Longform script segmentation into manageable production modules.
- Batch WAV rendering, clip review, targeted rerenders, and final stitching.
- Agent-guided production planning, transcript QA, and release evidence.
- Preserving useful human texture while keeping the copy clear.
