## Legends YT-DLP Slayer

> Clip faster. Create sharper.

Legends YT-DLP Slayer is a private Windows-first tool for pulling source video into a clean local workspace, checking what landed, and turning it into transcripts, searches, and clip plans.

Use it when a project starts with a channel, playlist, or hand-picked set of links and you want the boring parts handled: plan the job, keep output folders sane, verify files, search the words, and build cuts without babysitting every command.

The important part is the Mullvad lifecycle. Slayer is built to control the official Mullvad Windows app through its CLI: turn on the guarded production posture before capture, run the batch through that tunnel, then turn Lockdown off and disconnect Mullvad when the batch is done so the operator is not left stranded in VPN Lockdown mode.

## Best For

- Pulling source videos into predictable project folders.
- Building transcript and word-ledger searches from local media.
- Finding exact moments and turning them into FFmpeg clip plans.
- Checking source format, resolution, run reports, and finished files.
- Creator research, shorts, compilations, reviews, and remix prep.
- Windows-first operation with yt-dlp, ffmpeg, and Mullvad start-to-finish lifecycle controls.
- Default post-run cleanup: Lockdown off, Mullvad disconnected, final state verified.
