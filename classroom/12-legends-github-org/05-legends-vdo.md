## Legends VDO

> Persistent VDO.Ninja feeds for two-PC OBS capture, crop workflows, and reliable source restarts.

Legends VDO is for keeping capture senders ready when you want your work machine feeding clean video and audio into OBS on a second machine.

It wraps the VDO.Ninja sender workflow in a desktop app built for repeated recording setups: stable OBS Browser Source URLs, remembered feeds, native source sizing, optional crop-before-send, screen-crop mode for desktop overlays, and separate audio input/output feeds.

The latest release hard-restarts cropped sender webviews when capture settings change, so OBS does not keep receiving an old uncropped stream after the card and receiver URL have moved to the cropped output.

## Best For

- Two-machine OBS capture.
- Persistent sender workflows with stable receiver URLs.
- Window, screen, crop, and screen-crop feeds.
- Cleaner OBS sizing from native source bounds.
- Separate microphone/input and system playback audio feeds.
- Long recording setups where reliability matters.
