# The claude.md file

**The CLAUDE.md File (Brief Claude Once)**

CLAUDE.md is a file Claude reads automatically every session, so you stop re-explaining yourself. Put your brand voice, who you sell to, your offer, and your rules in it once. Run **/init** to generate a starter, trim it to the essentials, and whenever you correct Claude say "save this to memory."

> **Think of it this way:** It's onboarding a new hire. Write the brief once, and they show up knowing your business every day after.
