# Your first prompt

Talk to Claude Code like a teammate: say the outcome you want and any rules ("match my brand colors"). Before it touches anything, hit **Shift+Tab** to reach **Plan mode** - Claude researches and shows you a plan first, so you fix the direction before any work happens. Then approve, and choose whether it edits on its own or checks with you each step.

> Plan mode is reading the recipe before you turn on the stove. Cheaper to fix on paper than in the pan.
