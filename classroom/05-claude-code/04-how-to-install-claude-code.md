# How to install Claude Code

## **Install Claude Code (Do This Now)**

1. Make sure you have [Node.js 18+](https://nodejs.org/en) installed
2. Install [Claude Code](https://code.claude.com/docs/en/overview) globally
3. Verify the installation

**Windows users:** Use PowerShell or Windows Terminal, not the old Command Prompt. If npm isn't recognized, you need to install Node.js first from [**nodejs.org**](http://nodejs.org).

If you're not comfortable with the terminal, you can use IDEs like [VS Code](https://code.visualstudio.com/), [Antigravity,](https://antigravity.google/) [Cursor](https://cursor.com/), etc

[One command install vs code + claude code + extensions](https://github.com/AgriciDaniel/claude-code-essentials-vs-code)

## **Run Your First Command**

This is the moment. Open your terminal. Type `claude`. Hit enter.

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/a6eb024f3f9143d2b256e15115f43b35cb8ba42fa4cc41aaae23d8a596ae3201.png)

```
# Claude Code starts up. You'll see a prompt.

> What are the top 5 SEO mistakes small businesses make?

1. No keyword research - publishing content nobody searches for...
2. Ignoring technical SEO - slow pages, broken links, no sitemap...
3. Thin content - 300-word pages that add zero value...
4. No internal linking strategy - orphan pages everywhere...
5. Missing local SEO - no Google Business Profile...
```

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/2bdb032385d44d9d994fa024cb7224a4347f3f5274f94c9fbf51c2656f65889f.png)

If you got a response, congratulations. You now have the most powerful marketing CLI on the planet running on your machine. This is your foundation for everything that follows.

> **Don't overthink the first command.** Ask it anything. The point is to prove the tool works. In the next lesson, you'll install specialized skills that turn Claude Code into a full marketing department.
