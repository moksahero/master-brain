# MCP

**MCP (Plug In Your Tools)**

MCP connects Claude Code to outside tools and data — analytics, docs, services like DataForSEO — and Claude reaches for them when a task needs them. Manage them with **/mcp**: see what's on, switch off what you're not using. Every connected tool quietly eats memory even when idle, so keep only what you need.

> **Think of it this way:** They're browser tabs. A few open is fine; forty open and everything crawls.
