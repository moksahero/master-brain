# Agent view

**Agent View (One Control Tower)**

Once you're running more than one Claude Code session, **agent view** puts them all on a single screen — every job, every project, one place to watch. It's a research preview on Pro, Max, Team, Enterprise, and API plans.

> **Think of it this way:** Air-traffic control. One tower, every plane in view.
