# 🧰 Claude Code

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/cover-lvl-05.png)

These are the four skills that turn [Claude Code](https://www.skool.com/ai-marketing-hub-pro/classroom/708c1575?md=f579fae5e640447f90c73fb4ee8ed75f) into a marketing department. SEO, blog, paid ads, and a second brain. One install each, then you drive them with slash commands. Same engine, four jobs that used to need four hires and a stack of subscriptions.

> **Course promise:** Install a skill in one step, run it from a single command, and ship real work the same day. No menus, no glue code.

## What's inside

- [🔎 Claude SEO](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=0e87e1b9a1ea4e0ba7047bf57960939f) Turns Claude Code into a full SEO command center. Run `/seo` and parallel agents audit, fix, and optimize any site, then hand you a prioritized plan and a client-ready report.
- [✍️ Claude Blog](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=d36ed305e9dd4826852fe483df73fec8) Turns Claude Code into a full blog engine. One `/blog` command researches, drafts, adds schema, links internally, makes a hero image, and fact-checks every stat. Built to rank on Google and get cited by AI engines.
- [📣 Claude Ads](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=166ec4ea11f343648e30d0c3d471245c) Turns Claude Code into a paid-media audit room. One `/ads` command runs 250+ checks across 8 platforms and hands you one Ads Health Score with the top fixes ranked. Each finding has an ID, an impact, an owner, and an ETA.
- [🧠 Claude Obsidian](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=61647f28ea044ba694649a1a879b4104) Turns Claude Code into a self-organizing second brain. Drop in a source, it reads it, links the people and ideas, and files it into an Obsidian vault. Ask later and it answers from your own pages, with citations.

New here? Start with [🔎 Claude SEO](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=0e87e1b9a1ea4e0ba7047bf57960939f). It is the fastest way to a real first win.
