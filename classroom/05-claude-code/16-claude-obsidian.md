# 🧠 Claude Obsidian

Claude Obsidian turns [Claude Code](https://www.skool.com/ai-marketing-hub-pro/classroom/708c1575?md=f579fae5e640447f90c73fb4ee8ed75f) into a self-organizing second brain. You drop in a source. Claude reads it, pulls out the people and ideas, links them to what is already there, and files it into an Obsidian vault. You ask a question later and it answers from your own pages, with citations. Knowledge compounds.

> Install it, scaffold a vault with `/wiki`, ingest your first source, and query the vault like a research analyst. All from inside Claude Code.

![anim-real-claude-obsidian.gif](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/93961f65fe314c07936095587666e79cc3db72a44a084bd1af01c0f44e92876d.gif)

### Top commands

`/wiki` — Setup check, scaffold a vault, or continue where you left off

`ingest all of these` — Read all sources & creates linked pages,

`/save` — File the current conversation as a wiki note

`/autoresearch [topic]` — Autonomous research loop: search, fetch, synthesize, file

`/think [problem]` — Run the 10-principle thinking loop on a hard problem

`lint the wiki` — Health check: orphans, dead links, gaps, suggestions
