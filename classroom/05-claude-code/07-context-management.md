# Context Management

**Context (Why It Gets Dumber, And The Fix)**

Everything you do fills Claude's memory, and when it fills up, the answers get worse. Same task but slowing down? Run **/compact** to keep the gist and free up space. Starting something new and unrelated? Run **/clear** for a clean slate. Want to see what's eating memory? **/context**.

> **Think of it this way:** It's a whiteboard. /compact wipes it down to the summary, /clear wipes it blank. Don't start a new plan on yesterday's scribbles.
