**Slash Commands (Your Steering Wheel)**

You drive Claude Code with slash commands. The ones that matter:

**/init** — make a [CLAUDE.md](http://CLAUDE.md) from your folder

**/compact** — free up memory mid-task

**/clear** — fresh start

**/context** — see what's using memory

**/mcp** — manage connected tools

**/goal** — set an outcome for a long task

**/remote-control** — step away while it works

And **Shift+Tab** cycles permission modes: ask → auto-accept → plan.

> **Think of it this way:** Keyboard shortcuts. Awkward on day one, muscle memory by day seven.
