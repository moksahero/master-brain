# Explore, plan, code, commit

**Explore → Plan → Code → Commit (The One Rhythm)**

This is the loop for every task: let Claude **explore** (read and research), review its **plan**, let it **run** the work while it checks itself, then **review and ship**. Most people skip straight to "run" and pay for it in do-overs. Tell it exactly what "done" looks like and the first result usually lands.

> **Think of it this way:** Measure twice, cut once. The plan is the measuring.
