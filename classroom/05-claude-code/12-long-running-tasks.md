# Long running tasks

**Long-Running Tasks (Hand It Off, Walk Away)**

With Opus 4.8 you can give Claude Code a big, multi-step job and leave. Start it with **/goal** to set the outcome, then use **/remote-control** to step away and check in remotely. No babysitting — come back to finished work.

> **Think of it this way:** A dishwasher. Load it, press start, go live your life.
