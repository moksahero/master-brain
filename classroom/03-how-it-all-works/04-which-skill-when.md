# 🧭 Which Skill, When

> **What you'll be able to do:** Stop guessing. Match your exact situation to the exact command, and bookmark this lesson forever.

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/c3-which-skill-when.gif)

- **Have a site but don't know where you stand** — `/seo audit <url>` — Baseline scorecard
- **Aren't getting indexed / site feels broken** — `/seo technical <url>` — Crawl, index, speed, mobile
- **Need to find topics worth targeting** — `/seo cluster <keyword>` — Topic + keyword map
- **Need a deep research + strategy pass** — [**Marketing Brain**](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=888db513f2b24f4d85320fe80ee88bbb) — 6-step [DataForSEO](https://www.skool.com/ai-marketing-hub-pro/classroom/2afc71bf?md=af828c1dd05c40aeb37ff7fe01730f13) pipeline
- **Need to write content** — `/blog write <topic>` — Drafts optimized content
- **Have content but a page won't rank** — `/seo content <url>` — On-page optimization
- **Want Google to understand the page** — `/seo schema <url>` — Structured data
- **Want ChatGPT / AI Overviews to cite you** — `/seo geo <url>` + `/blog geo` — AI citability
- **Run a local or service-area business** — `/seo local` · `/seo maps` · [**Local SEO Brain**](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=22902679826f428295b66ca4141160bd) — Local rankings
- **Want to multiply a winning topic** — `/seo cluster` · `/blog cluster` — Build the hub
- **Need to know if it's working** — `/seo google report full` — Progress + standing
- **Worried rankings slipped** — `/seo drift compare <url>` — Catch regressions
- **Have a page that ranks AND converts** — `/ads audit` — Amplify with paid
- **Want strategy that compounds** — **Marketing Brain** — Persistent strategy vault

---

- Every situation maps to one command. Find yours in the table.
- Audit → technical → research → write → optimize → structure → geo → scale → amplify.
- When unsure, start with `/seo audit`.
