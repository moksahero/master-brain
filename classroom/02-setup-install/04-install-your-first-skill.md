# 🔌 Install Your First Skill

> Install any skill in the Hub by clicking through a menu. No terminal, no commands, no weird slugs. If you can use a settings screen, you can do this.

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/term-install-skill.gif)

*Install by plain English. No slugs to memorize.*

**Prompt your claude:**

```
install this skill from https://github.com/AI-Marketing-Hub/claude-seo and check we're up to date. Also, check that my github cli is connected as it is a private repo and only members have access. 
```

Most install guides throw terminal commands at you. You don't need them. [Claude Code](https://www.skool.com/ai-marketing-hub-pro/classroom/708c1575?md=f579fae5e640447f90c73fb4ee8ed75f) has a **built-in menu** that does it with clicks.

![seo-command-demo.gif](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/22e9815500e44e169c6edcafa325e6df4a7f52c4cd02487c9a0d1fed7c2d0da7.gif)

**You're done when:** the skill shows under **Installed** and its command responds.

> 💡 This is the path to hand mid-technical teammates and clients. They never see a slug or a terminal. They pick from a list and click.

## The command way (only if you prefer typing)

Same result in two lines:

```text
/plugin marketplace add AI-Marketing-Hub/claude-seo
/plugin install claude-seo@ai-marketing-hub-claude-seo
```
