# 🧠 Install the Brains

> Set up [Obsidian, so ](https://obsidian.md/)your skills read from your own machine.

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/term-install-brain.gif)

*Clone and set up everything in one go.*

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/2b6964ecc39e46a9b8be6fdd9688715e8e1ddaa1c1814db1a258f652d04f5328-md.png)

The skills are plugins. The [**Brains**](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=88a46ed175f747279433faaa92354614) are folders you clone and run locally. They're optional power-ups, not day-one requirements. Grab them when a skill asks for one, or when you want the visual dashboards.

---

## The Brains (Obsidian vaults that give your work memory)

[**Local SEO Brain**](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=22902679826f428295b66ca4141160bd) is a pure maps & gmb ranking template.

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/e720c47d83d74506a1d3795336dc7f873e478e8a164e4a548d861307da9e2395.png)

[**Marketing Brain**](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=888db513f2b24f4d85320fe80ee88bbb) is the 6-step [DataForSEO](https://www.skool.com/ai-marketing-hub-pro/classroom/2afc71bf?md=af828c1dd05c40aeb37ff7fe01730f13) research pipeline.

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/152bc9ccd99a48199b1337a39e99f55badf75fedb0e244b68fdfdc5a76c18472.png)

[**Ads Brain**](https://github.com/AI-Marketing-Hub/ads-brain)** **is a power up to your** **[**claude ads**](https://github.com/AI-Marketing-Hub/claude-ads)**.**

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/7487ecf1e21d4021bcc314bb34916e87b5e5108f719540e6b40783dd4f0f4341.png)

[**Website Brain**](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=1dc04a82c97547d1aa66353d805793b5) *builds* a brain from any site you crawl. It is a [Claude Code](https://www.skool.com/ai-marketing-hub-pro/classroom/708c1575?md=f579fae5e640447f90c73fb4ee8ed75f) plugin that runs on [Firecrawl](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=0c17dddab4274ddaa33b75e2855ede57).

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/31229e71f7b44f26870e447cb07ac3033c17bdfa08e347bd887885ad17b8841f.png)

> ✅ Clone **one** Brain that matches your path and open it using [Obsidian](https://obsidian.md/). Post a screenshot of the vault graph (Brains). Stuck? Drop the error.

- Brains are Obsidian vault templates (Local SEO & Ads need no Python; Marketing Brain adds a Python pipeline).
- All of it runs on your machine with your own keys. Keep them in `~/AI-Marketing-Hub`.
