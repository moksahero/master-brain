![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/cover-lvl-02.png)

> Get the tools you need installed and working, understand where everything lives, and know how to update it later, without touching anything scary.
