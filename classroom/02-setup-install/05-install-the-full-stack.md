# 📦 Install the Full Stack

> Get all four core skills installed the easy way. Repeat the click flow, or drop one file to install them all at once.

---

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/demo-blog-install.gif)

## ⚡ Easiest, one prompt to claude (whole stack at once)

**Prompt your claude:**

> "Hey, I want you to install some skills for me from a private GitHub org. Before anything else, here's the setup work:
> 
> **Check my GitHub CLI connection.** Run `gh auth status` and confirm I'm authenticated. If I'm not connected, stop and walk me through running `gh auth login` first — these are private repos, so this step is required.
> 
> **Verify access to the org.** Once auth is confirmed, check that I can reach the private repos under [https://github.com/AI-Marketing-Hub](https://github.com/AI-Marketing-Hub) (e.g. run `gh repo view AI-Marketing-Hub/claude-seo`). If any repo comes back as not found or access-denied, tell me which ones so I can sort out permissions before we continue.
> 
> **Install these four skills** (clone or add as marketplaces/plugins, whichever the setup expects):
> 
> [https://github.com/AI-Marketing-Hub/claude-seo](https://github.com/AI-Marketing-Hub/claude-seo)
> 
> [https://github.com/AI-Marketing-Hub/claude-blog](https://github.com/AI-Marketing-Hub/claude-blog)
> 
> [https://github.com/AI-Marketing-Hub/claude-obsidian](https://github.com/AI-Marketing-Hub/claude-obsidian)
> 
> [https://github.com/AI-Marketing-Hub/claude-ads](https://github.com/AI-Marketing-Hub/claude-ads)
> 
> When you're done, give me a quick summary: auth status, which repos you could access, and which skills installed successfully."

---

#### /Plugin way: repeat the menu for each skill

In the `/plugin` menu → **Marketplaces** tab, add each repo below; then in **Discover**, click **Install** on each. No slugs, no terminal.

**SEO** — `AI-Marketing-Hub/claude-seo` — claude-seo

**Blog** — `AI-Marketing-Hub/claude-blog` — claude-blog

**Ads** — `AI-Marketing-Hub/claude-ads` — claude-ads

**Obsidian** — `AI-Marketing-Hub/claude-obsidian` — claude-obsidian
