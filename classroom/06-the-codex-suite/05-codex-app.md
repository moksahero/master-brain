# Codex App

**Meet the Codex App (Your Command Center)**

The Codex app gives you one place to run and supervise agents instead of juggling terminal windows — your projects on the left, a live conversation on the right. Dictate a task by voice, watch Codex work, then review the changes, leave comments, and approve when it looks right. Plug in your tools with skills, run several agents at once, and hand long jobs to the cloud.

> **Think of it this way:** Mission control. Every agent and project on one screen — you just supervise.

**Coming from Claude Code?** It's the GUI cousin of the terminal, same agent underneath.
