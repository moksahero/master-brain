# 🔎 Codex SEO

> Run a site audit, deep-dive a single page, and produce optimized content from Codex CLI. This is the Claude SEO command center, Codex-native.

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/cover-real-codex-seo.png)

### What codex-seo does

codex-seo is the Codex port of the Hub's SEO skill. Audits, technical checks, content quality, schema, local, and AI visibility (AI Overviews and ChatGPT). The orchestrator runs parallel agents and returns a 0 to 100 score with a fix list ordered by impact. Same as the Claude version.

### Install it

```bash
codex plugin marketplace add AI-Marketing-Hub/codex-seo
codex plugin add codex-seo@ai-marketing-hub-codex-seo
```

Confirm with `codex plugin list`. (Full install pattern in [Lesson 2](https://www.skool.com/ai-marketing-hub-pro/classroom/ded6ca5c?md=b6f648a53a404b369356923761262dba).)

### Run it

Once installed, ask Codex to audit a site. The output is the same client-grade report you get in the Claude SEO course: a score, the issues that move rankings, and a fix list ranked by impact.

```text
Run a full SEO audit on https://yoursite.com. Return the overall score,
the top 5 fixes ranked by ranking impact, and flag any AI-visibility
gaps for AI Overviews and ChatGPT.
```

A single-page deep dive:

```text
Deep-dive https://yoursite.com/pricing for on-page SEO: title, meta,
headings, internal links, schema, and content depth. Give me a
prioritized fix list I can hand to a writer.
```

It gets sharper when you connect [DataForSEO](https://www.skool.com/ai-marketing-hub-pro/classroom/2afc71bf?md=af828c1dd05c40aeb37ff7fe01730f13) for live keyword and rank data, plus Search Console for indexation truth. Without them you still get a strong technical and on-page audit.
