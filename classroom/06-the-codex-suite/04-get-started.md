# Get started

This video is the full tour — it covers both the Codex app and the command line, start to finish. To get running right now: install the CLI with one command and launch with `codex`, or download the Codex app and sign in. Either way, you sign in with your ChatGPT account.

```
curl -fsSL https://chatgpt.com/codex/install.sh | sh   # CLI (macOS / Linux)
codex                                                  # launch in your project
```

> **Think of it this way:** The app is the dashboard, the CLI is the cockpit. Same engine — pick your seat.

**Coming from Claude Code?** Same "run it in your project" flow; just swap the install URL.  
**Read more →** [https://developers.openai.com/codex/cli](https://developers.openai.com/codex/cli)
