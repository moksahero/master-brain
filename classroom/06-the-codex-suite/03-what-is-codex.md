# What is Codex

**What Is **[**Codex**](https://www.youtube.com/watch?v=HFM3se4lNiw)**? (An Agent That Checks Its Own Work)**

Codex is OpenAI's coding agent that lives in your terminal, IDE, and app — it's not a chatbot. You give it a task and it plans, does the work, then *validates itself*: runs tests, launches the app, checks the logs. So when it says "done," it's usually done — not a pile of errors waiting for you.

> **Think of it this way:** A contractor who doesn't just build the deck — they jump on it a few times before handing you the keys.

**Coming from Claude Code?** Same agentic loop, different engine.
