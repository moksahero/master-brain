# /goal

**Hand Off Big Jobs (Goals)**

Type `/goal`, state a concrete, checkable outcome, and Codex works toward it — for hours or even days — deciding for itself when it's done. Steer it mid-run, peek at progress in a side chat without interrupting, and pause or resume when you step away. The trick: make the goal something it can *verify* (a target hit, tests passing).

> **Think of it this way:** A slow cooker. Set a clear result, walk away, come back to something finished.

**Coming from Claude Code?** Same idea as `/goal` plus remote hand-off.
