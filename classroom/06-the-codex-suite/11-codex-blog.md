# ✍️ Codex Blog

> Produce a ranking-ready post, rewrite a weak one, and audit live content for E-E-A-T and AI citations from Codex CLI. This is the Hub's blog engine, Codex-native.

### What codex-blog does

codex-blog is the full blog lifecycle engine ported to Codex. The whole pipeline: strategy, topic clusters, briefs, outlines, drafting, rewriting, audits, schema, AI-citation optimization (GEO/AEO), and repurposing. It's optimized for Google rankings and AI citations, and works on any platform: WordPress, Next.js MDX, Hugo, Ghost, and more.

### Install it

```bash
codex plugin marketplace add AI-Marketing-Hub/codex-blog
codex plugin add codex-blog@ai-marketing-hub-codex-blog
```

### Run it

Start anywhere in the pipeline. Plan a cluster, write a fresh post against a brief, or audit something live and rewrite the weak parts. The scoring is the same 5-category, 100-point system as the Claude version, so a "good" score means the same thing.

Audit and rewrite an existing post:

```text
Audit https://yoursite.com/blog/my-post for E-E-A-T, AI-citation
readiness, and on-page SEO. Score it, then rewrite the weakest two
sections to lift the score, keeping my voice.
```

Plan a topic cluster:

```text
Build a topic cluster around "email marketing for SaaS": one pillar
page plus 6 supporting posts, with target keywords and an internal-link
structure. Return it as a publishing plan.
```

### Same as the Claude version

The big lever is repurpose. One strong post becomes a newsletter, a thread, and social snippets. Write once, distribute everywhere. The methodology matches Claude Blog. For the templates, the FLOW framework, and the full pipeline, see the product course.
