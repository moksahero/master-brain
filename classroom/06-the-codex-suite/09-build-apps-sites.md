# Build apps sites

**Build & Share Apps (Sites)**

Codex can turn an idea into a real, hosted app and publish it in minutes - hosting, login, storage, and a database all included, no deploy code. Think dashboards, onboarding hubs, event-prep pages, an interactive report, anything that used to be a doc or a slide. Share it by link; people open it by signing in with ChatGPT.

> **Think of it this way:** Describe the dashboard out loud, get a link to it. Slides that actually do something.
