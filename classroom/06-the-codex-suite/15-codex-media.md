# 🎬 Codex Media

> Turn a long video into shorts, clip the best moments from a YouTube link, generate images, and build GIFs from Codex CLI. This is the Hub's whole media toolkit, unified into one Codex skill.

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/cover-real-codex-media.png)

### What codex-media does

codex-media is a unified media suite ported to Codex. Five jobs in one skill:

- **Video** - edit, caption, enhance, and export.
- **Shorts** - extract vertical, viral-ready clips from long footage.
- **YouTube** - pull and clip the best segments from a YouTube URL.
- **Images** - generation and edits.
- **GIFs** - create, edit, and optimize.

### Install it

```bash
codex plugin marketplace add AI-Marketing-Hub/codex-media
codex plugin add codex-media@ai-marketing-hub-codex-media
```

### Run it

This is where one asset becomes ten. Record one long video and codex-media gives you the shorts for Reels and TikTok, the YouTube clips, the captioned cut for LinkedIn, and a GIF for the newsletter. Same conversational workflow as the Claude video and shorts skills. You describe what you want, it does the cutting and rendering.

Turn a long video into shorts:

```text
Take this 20-minute video and pull the 3 best short clips for
TikTok and Reels. Vertical 9:16, animated captions, 30-60 seconds
each, hook in the first 2 seconds.
```

Clip a YouTube link:

```text
From this YouTube URL, find the 5 most quotable moments and export
them as captioned vertical clips ready for social.
```

### Same as the Claude version

The fastest win is shorts. Hand it a long video, it transcribes, finds the high-retention moments, and renders vertical clips with animated captions sized for each platform. That alone replaces hours of manual editing. Same conversational, no-menu approach as the Claude video and shorts skills.
