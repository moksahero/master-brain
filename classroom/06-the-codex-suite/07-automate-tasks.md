# Automate tasks

**Automate the Boring Stuff (Scheduled Tasks)**

Codex can run tasks on a schedule, in the background, while you focus elsewhere. Set up a morning digest of what happened yesterday, a recurring report, a monitor that triages issues, or an overnight job that keeps your [AGENTS.md](http://AGENTS.md) and skills sharp. Automations have memory, so they don't repeat the same work twice.

> **Think of it this way:** A night-shift teammate. You sleep, it works, you wake up to it done.
