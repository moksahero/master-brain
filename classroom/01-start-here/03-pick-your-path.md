# 🧭 Pick Your Path

### 🔎 Path A: Rank (you want traffic and clients from search)

**Start with:** [Claude SEO](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=0e87e1b9a1ea4e0ba7047bf57960939f). **Your first move:** install it (next course), then run an audit on your own site. **Then:** follow [The Client Delivery Flow](https://www.skool.com/ai-marketing-hub-pro/classroom/d2605b58?md=89d92e3fde4e4320acdfd6f9c1bbb583) to turn that into client work.

---

### ✍️ Path B: Publish (you want to produce content at scale)

**Start with:** Claude Blog. **Your first move:** install it, then write one optimized post on a topic you know. **Then:** the [Claude Blog](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=d36ed305e9dd4826852fe483df73fec8) folder for clusters and the quality gates.

---

### 📣 Path C: Advertise (you run or manage paid ads)

**Start with:** Claude Ads. **Your first move:** install it, then audit one ad account. **Then:** the [Claude Ads](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=166ec4ea11f343648e30d0c3d471245c) folder for the 8-platform playbook.

---

### Not sure? Two tie-breakers

- **Have a website you want to grow?** → Path A (Rank).
- **Already spend on ads?** → Path C (Advertise).
- Still stuck? Default to **Path A**. Search traffic compounds and feeds everything else.

> > **✅ **Bonus lanes once you're moving: [**Claude Obsidian**](https://www.skool.com/ai-marketing-hub-pro/classroom/b5886e72?md=61647f28ea044ba694649a1a879b4104) to organize it all.
