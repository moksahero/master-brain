# 🎯 The Client Delivery Flow

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/cover-lvl-04.png)

The whole point of the tools is delivering work people pay for. This course is one repeatable flow you run on every client: plan, analyze, draft, edit, improve. Same five steps every time. The tools just slot into the steps.

> **Course promise:** Take a project from "new client" to "delivered and reporting" using one flow you can run again and again, with the exact command at each step.
