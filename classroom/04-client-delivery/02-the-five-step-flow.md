# 🔁 The Five-Step Flow

> Run one repeatable flow on every client: plan, analyze, draft, edit, improve. Same five steps, every time.

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/chat-cf-five-step.gif)

*Ask for the flow and Claude pulls the same five steps you run on every client.*

The tools are only worth money when they deliver client work. So here's the flow you run on every project. It never changes. The tools just slot into the steps.

---

### The five steps

1. **Plan** · scope the client. What they sell, who they want, what "win" means, and getting access.
2. **Analyze** · audit what's there and research the opportunity. `/seo audit` + [Marketing Brain](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=888db513f2b24f4d85320fe80ee88bbb).

  ![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/14bd7aca99e64da1adb33bcaadd45f493b28033bde154ce68d123d4ba4973dcb.png)
3. **Draft** · make the work. Content, pages, ad campaigns. `/blog write`, `/ads`.
4. **Edit** · review and QA before anything ships. Fact-check, brand, quality gates.
5. **Improve** · report what happened and set the next cycle. `/seo drift`, `/seo google report full`.

---

### Map it to the folder

This is exactly the client folder from the last course. Each step writes into it:

- Plan → `notes.md`
- Analyze → `audits/` + `research/`
- Draft → `content/`
- Edit → the same files, reviewed
- Improve → `reports/`

> ✅ Post the five steps in your own words, and which step you usually skip. (Most people skip Analyze or Edit. Naming it fixes it.)

- One flow, every client: Plan → Analyze → Draft → Edit → Improve, then loop.
- A fixed flow makes you faster, stops skips, and is handoff-able.
- Each step writes into the client folder. The loop is the retainer.
