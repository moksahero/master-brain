# `Quick Start Checklist`

#### Follow these steps to get the workflow running.

**1. Get Your Tools Ready (Prerequisites)**

✅ **Install n8n Desktop:** Use the link above if you haven't already.

✅ **Get API Keys:** This workflow requires accounts (some paid).

✅ **Prep Your Google Sheet:** Make a copy of the template below to use for your output.

[https://docs.google.com/spreadsheets/d/1M9Pc6xKRpbz6kV4VsPkGZXhJIyccFVtVaBEzBFBx1ik](https://docs.google.com/spreadsheets/d/1M9Pc6xKRpbz6kV4VsPkGZXhJIyccFVtVaBEzBFBx1ik)

**2. Download & Import**

✅ **Download the Workflow File** attached to this lesson.

✅ **Import the **`.json` file into your n8n Desktop App.

**3. Configure the Workflow**

✅ **Add Your API Keys** and Google credentials into the marked nodes in the workflow.

✅ **Update the Google Sheet ID** in the Google Sheets nodes to point to your copy.

*Refer to the main video for a detailed walkthrough of this step.*

**4. Run It!**

✅ Activate the workflow and use its webhook URL to access the start form. Fill it out to generate your first article.

# `Troubleshooting Tips`

**Error? Check Credentials First.** 90% of issues come from an incorrect API key or Google Sheet ID. Double-check everything.

**Workflow Outdated?** This was built with a specific n8n version. If you run into major issues, you may need to adjust nodes to work with the latest n8n updates. This workflow is best for users comfortable with troubleshooting.

**Still Stuck? Ask the Community.** Post a screenshot of the node that has an error in the `Ask for Help❓` section.
