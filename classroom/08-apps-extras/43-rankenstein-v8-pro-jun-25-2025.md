# `What This Workflow Does`

![image.png](https://assets.skool.com/f/08e74c5b924e48ea82902d53df2be280/f823b5af820d4b7cb302d3dc1a66a1f5af77181d3aa943de98296fa80de9e3af)

This is a focused research powerhouse:

`Deep Research`**:** Uses a multi-agent system to pull and analyze data from Tavily and Perplexity.

`Actionable Strategy`**:** Identifies competitor weaknesses and develops a keyword strategy with Core Topics, Long-Tail Queries, and Pain Points.

`Universal Output`**:** Delivers the research report and first draft as clean **.docx** and **.html** files via email, so you can use them on any platform (WordPress, Webflow, etc.).

# `Quick Start Checklist`

Follow these steps to get your research engine running.

**1. Read the Official Guide (Required)**

✅ **Start with **`Rankenstein v8 Pro Research Setup Guide.pdf`. This is your main manual and details the API setup.

**2. Get Your Tools Ready (Prerequisites)**

✅ Ensure you have an active n8n instance (Cloud or Self-Hosted).

✅ Based on the PDF guide, get your API keys for **Gemini, Tavily, and Perplexity**.

**3. Import, Configure and Run**

✅ **Import the **`Rankenstein_v8_Pro_Research.json` file into your n8n instance.

✅ Following the PDF guide, **add your API credentials** into the correct nodes.

✅ Activate the workflow and use the native n8n form trigger to start your first research report.

# `Troubleshooting`

**The PDF Guide is Your First Stop.** 99% of setup questions are answered there.

**Check All API Keys.** The most common errors come from incorrect credentials. Carefully re-check every connection.

**Workflow Updates:** This is an advanced workflow. Future updates to n8n or the connected APIs may require you to troubleshoot and modify nodes.
