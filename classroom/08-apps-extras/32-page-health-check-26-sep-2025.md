# Page Health Check (26 Sep 2025)

> This workflow runs a rapid "physical" on your landing pages. It triggers via a simple form submission and returns a report on how your page is performing technically.

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/b3146e826391418da397144a7b8d12137d4f079c6e194822b48e959d75b5e831-md.png)

**⚡ What It Does (**[**Example**](https://pub-6e4b77b888bc4422963711caabeb1953.r2.dev/research.html)**):**

**🔎 Quick Analysis:** Scans your URL instantly.

**📊 Technical Metrics:** Checks loading speeds and core web vitals.

**🤖 Tech Stack:** DataForSEO + n8n + Gemini

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/1177dd10255d48e3951c3c66ecf0d6d6f6601c94235b4441b5068bad2012ff8d.png)

**How to use:**

1. Import the JSON file below.
2. Open the "Form Trigger" node to get your submission URL.
3. Send a test URL to see the health report.
