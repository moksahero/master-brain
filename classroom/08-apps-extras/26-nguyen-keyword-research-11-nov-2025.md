# Nguyen Keyword Research (11 Nov 2025)

> Built by **Nguyen**, this workflow adds an intelligent caching layer to DataForSEO. It checks your internal database *before* spending money on API calls.

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/49429e3e2586475783d324a6488fcc8bd955db8564d44c4683e145e18981c3b6-md.png)

**Stop Wasting API Credits.** Debugging SEO workflows can get expensive fast. This smart workflow solves that problem by implementing a "Cache Hit" system.

**⚡ How It Works:**

1. **Check Cache:** Before calling the API, it checks your n8n Data Table.
2. **Skip Cost:** If the keyword exists, it loads it instantly (Free).
3. **Fetch New:** If not, it calls DataForSEO and saves the result for next time.

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/d6839af1c736486bafa59ce38784dd12cf507ddee5ed408ebb7646d8a1b4d048-md.png)

**🚀 The Features:**

**💾 Smart Caching:** Never pay for the same keyword twice.

**🧹 JSON Cleaner:** Restructures messy API data into a clean object.

**📊 HTML Report:** Generates an interactive report with filters and charts.
