# 🧠 Brainstein

![brainstein-header.webp](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/f9abef3571cf4f92aa92cadcc4eb6fa44278993449054b2c9cc357ab223d252b)

We ship brains for SEO, Ads, and Blog. Brainstein is the generator that builds you one for any domain we do not cover: accounting, sales, research, ops, content. You write a short YAML spec, and Brainstein scaffolds a complete second-brain repo: vault, agent skill pack, source ledger, demo, tests, and a 33-item audit.

> **Course promise:** Install Brainstein, write a starter spec, generate a tiny demo brain from it, and run the audit to see your maturity score.

Advanced and optional. This is a build-your-own-brain tool, not a day-one requirement. Reach for it only when you need a brain for a domain we do not ship.

> Where the apps are the "see" layer, Brainstein is one level deeper: it generates the brains the skills read from.
