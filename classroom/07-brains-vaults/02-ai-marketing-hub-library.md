# 📊 AI Marketing Hub Library

![image.png](https://assets.skool.com/f/49ff1f2d656742a68e4b871cd8c0e543/40583ce04afd49f8a85f0b5e344bb1761e4b9c2121c94b4288f5283483b24510-md.png)

> AI Marketing Hub Library is the ultimate orchestrator Brain. It is the first thing a member sets up: hook it to [Claude Code](https://www.skool.com/ai-marketing-hub-pro/classroom/708c1575?md=f579fae5e640447f90c73fb4ee8ed75f) or [Codex](https://www.skool.com/ai-marketing-hub-pro/classroom/ded6ca5c?md=ed1717deecdb4decb7d5bd674c0f030d), state a marketing goal, and it routes the work into the exact Hub workflow with the prompts, prerequisites, and deliverable format already loaded. It is a Brain, not a prompt folder. Every other tool and Brain plugs into it.

**What this Brain does:** Routes a goal to one of 8 outcome workflows: best-seo-performance, content-engine, paid-amplify, local-domination, client-delivery-loop, authority-content, creator-repurpose, or research-to-strategy. It carries 259 reference prompts, a 614-note Obsidian library, and the `/start`, `/goal`, and `/campaign` command center. It checks prerequisites and refuses fabricated data or pasted secrets.
