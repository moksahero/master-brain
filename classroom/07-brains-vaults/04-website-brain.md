# 🌐 Website Brain

![](https://cdn.jsdelivr.net/gh/AI-Marketing-Hub/classroom-assets@cfc5a1e99148d256bff96912f16ee28322523f56/cover-real-website-brain.png)

> 🎬 **See it built live (full walkthrough):** [Website Brain: turn any site into a vault](https://www.skool.com/ai-marketing-hub-pro/website-brain-turn-any-site-into-a-vault)

Website Brain is the capture brain. Point it at any site (yours, a client's, or a prospect's) and it crawls the whole thing with Firecrawl, then builds a self-contained Obsidian brain: one clean note per page with a full-page screenshot, an organized image library, the site's design DNA distilled into a generation-ready `brand-tokens.json`, a wikilink graph that mirrors the site, and an adversarial quality review.

> **What this Brain does:** Turns a whole website into a sourced, offline brain that both you and your AI agents can read. Crawl once, then work fully offline.
> 
> Want to scaffold a brand-new brain for a different domain instead of capturing a site? That is [Brainstein](https://www.skool.com/ai-marketing-hub-pro/classroom/c1d53bcf?md=96488a5da2a74801bc83da74aafa756e). Website Brain captures an existing site. Brainstein builds a brain from a spec.
