# 🧰 Master Brain Commands

> Repo-owned bucket (not from the captured classroom). The blessed, runnable
> `/mb:` commands for installing, onboarding, operating, and tracking the AI
> Marketing Hub. Each is a slash command — keep it, fill any `<brackets>`, run.

## Prompts

**Not sure what to do — route me:**

```text
/mb:idk <what you're trying to do, e.g. rank my plumbing business in Austin>
```

**Install the whole fleet (brains + claude-ads + claude-mem):**

```text
/mb:install
```

**Scaffold a project (asks your goal, wires the right brains, queues first TODOs):**

```text
/mb:init <project name or goal>
```

**Health check — what's installed, API keys, tooling, todo backlog:**

```text
/mb:doctor
```

**Update every installed brain to the latest version:**

```text
/mb:update
```

**Run the fused multi-brain client intelligence report for a site:**

```text
/mb:report <https://client-site.com>
```

**Promote this project's tooling improvements up to the source repo:**

```text
/mb:push
```

**Capture a follow-up into the tracked backlog:**

```text
/mb:todos-add "<the follow-up to remember>"
```

**See the backlog (open / done / all):**

```text
/mb:todos-list
```

**Triage the backlog — dedupe, prioritize, mark stale/blocked:**

```text
/mb:todos-review
```

**Work every open TODO to completion (records an outcome on each):**

```text
/mb:todos-execute
```

**Generate recurring TODOs on a cadence (pair with /schedule):**

```text
/mb:todos-routine
```

**Project history — what's been done, chronologically, with outcomes:**

```text
/mb:todos-log
```
